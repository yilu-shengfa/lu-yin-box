## 1. 软件概述

| 项目 | 信息 |
|------|------|
| **软件名称** | 鹿音盒（LuYin Music Box） |
| **当前版本** | v1.8.87 |
| **包名** | tv.ylsf.box.mobile |
| **支持平台** | Android 5.0（API 21）及以上 |

### 简介

鹿音盒是一款功能丰富的 Android 音乐播放器应用，提供高品质音乐播放、精美歌词显示、专业级音效调节、本地音乐管理、数据云端同步等全方位音乐体验。软件界面简洁优雅，操作流畅自然，致力于为用户打造一款真正好用的移动端音乐工具。

## 2. 快速入门指南

### 第一步：安装应用

从 [Releases] 页面下载对应架构的 APK 安装包并安装。首次启动时，请允许应用请求的必要权限（存储权限、通知权限等），以确保各项功能正常运行。

### 第二步：搜索并播放音乐

1. 打开应用后，点击顶部的「立即搜索」入口
2. 输入您想听的歌曲名称或歌手名
3. 选择一个搜索结果，点击即可开始播放

### 第三步：探索更多内容

通过右/左侧导航菜单（横屏模式）或顶部导航栏（竖屏模式），您可以访问：

- **在线歌单** — 浏览热门歌单和精选推荐
- **排行榜单** — 查看各大平台音乐排行榜
- **每日推荐** — 根据您的喜好获取个性化推荐
- **本地音乐** — 扫描并播放手机中的本地音乐文件

### 第四步：个性化设置

点击导航菜单中的「软件设置」，您可以：

- 切换主题颜色和外观模式
- 调整字体大小以适应阅读习惯
- 配置桌面歌词、音效等高级功能
- 管理下载路径和数据同步

### 第五步：享受音乐

现在您已经掌握了基本用法！在播放详情页面可以查看歌词、调整音效、切换播放模式等。尽情享受音乐吧！

## 4. 音乐播放功能

### 多源搜索

鹿音盒支持通过多种音乐源进行搜索和播放，不同音源可用的音质选项可能有所不同。

### 播放控制

| 功能 | 操作方式 |
|------|----------|
| **播放 / 暂停** | 点击底部播放栏中央按钮或播放页封面区域 |
| **上一曲 / 下一曲** | 点击播放控制区的上/下一曲按钮 |
| **进度条拖拽** | 左右拖拽进度条跳转到指定播放位置 |
| **定时停止** | 设置定时器，到时自动停止播放（支持多种时长选择） |

### 播放模式

支持以下六种播放模式，点击循环模式按钮依次切换：

1. **列表循环** — 播放完列表最后一首后回到第一首继续播放
2. **顺序播放** — 按列表顺序依次播放，播完即停
3. **随机播放** — 随机打乱顺序播放
4. **单曲循环** — 反复播放当前这一首歌曲
5. **禁用切换** — 锁定当前播放模式，防止误触切换
6. **心动模式** — 智能优先播放您标记为"喜欢"的歌曲

### 音质选择

鹿音盒提供多档音质供您选择：

| 音质等级 | 说明 |
|----------|------|
| **128K** | 标准 MP3 音质，节省流量和存储空间 |
| **320K** | 高品质 MP3 音质，适合大多数场景 |
| **FLAC** | 无损压缩格式，保留完整音质细节 |
| **更高音质** | 视音乐源支持情况提供更高级别的音质选项 |

> 注意：高音质选项需要对应音乐源支持，且会消耗更多网络流量和存储空间。

### 其他播放功能

- **歌曲换源** — 当前歌曲无法播放时，一键切换到其他音源尝试播放
- **播放历史** — 记录最近播放过的歌曲，方便回溯收听
- **MV 播放** — 支持观看歌曲的官方 MV 视频
- **分享歌曲** — 将喜欢的歌曲分享给好友
- **稍后播放** — 将歌曲加入"稍后播放"队列
- **上下滑动切歌** — 在播放详情页上下滑动屏幕即可快速切换歌曲

## 5. 歌词功能详解

鹿音盒拥有强大而精美的歌词系统，满足从日常欣赏到专业制作的各种需求。

### 桌面歌词悬浮窗

桌面歌词可在任何应用上层显示，让您在使用其他应用时也能看到当前播放歌曲的歌词：

| 设置项 | 说明 |
|--------|------|
| **字体大小** | 调整悬浮窗歌词的文字大小 |
| **透明度** | 调整悬浮窗背景的透明程度 |
| **位置** | 自由拖拽调整悬浮窗在屏幕上的位置 |
| **最大行数** | 控制同时显示的歌词行数 |
| **单行模式** | 仅显示当前正在演唱的一行歌词 |
| **锁定** | 锁定位置后防止误触移动 |
| **主题色** | 选择歌词的高亮主题颜色 |

### 播放页歌词

播放详情页内置两种歌词布局方式：

#### 水平布局
- 歌词与封面左右并列显示，适合横屏使用

#### 垂直布局
- 歌词与封面上下排列显示，适合竖屏手持操作

**可调参数：**

- **字体大小** — 调整歌词文字大小
- **对齐方式** — 左对齐 / 居中对齐 / 右对齐
- **宽度调节** — 调整歌词区域的宽度占比

### KRC/KSC 逐字歌词

支持 KRC 和 KSC 格式的逐字时间轴歌词，呈现卡拉OK式的逐字高亮效果——每个字随着演唱进度逐个变色点亮，带来沉浸式跟唱体验。

### 双语歌词

同时显示原文歌词和翻译歌词，帮助理解外语歌曲的含义。

### 自定义歌词背景

让您的歌词显示更加个性化：

- **本地图片** — 从相册中选择本地图片作为歌词背景
- **网络图片** — 输入图片 URL 使用网络图片作为背景
- **透明度** — 调节背景图的透明程度
- **模糊度** — 调节背景图的模糊效果（毛玻璃效果）

### 蓝牙歌词

支持通过蓝牙将歌词推送到车载音响、智能手表等支持蓝牙歌词显示的设备上。

### 歌词拖拽调整进度

在播放详情页长按歌词区域并左右拖拽，可以直接调整播放进度——拖到哪句歌词就从哪里开始播放。

## 6. 歌词高级功能

### 导入本地歌词

支持通过 SAF（存储访问框架）文件选择器导入以下格式的本地歌词文件：

- `.lrc` — 标准 LRC 格式歌词
- `.krc` — 酷狗 KRC 加密格式歌词
- `.txt` — 纯文本歌词
- `.trc` — 其他歌词格式

### 智能自动校正

导入歌词后，系统会自动检测歌词的时间标签是否存在偏移（即歌词显示快了或慢了），并给出修正建议。您可以选择接受建议或手动微调。

### 手动调整偏移

提供五档精细偏移调节：

| 按钮 | 效果 |
|------|------|
| **-1s** | 歌词整体提前 1 秒 |
| **-100ms** | 歌词整体提前 100 毫秒 |
| **重置** | 恢复默认无偏移状态 |
| **+100ms** | 歌词整体延后 100 毫秒 |
| **+1s** | 歌词整体延后 1 秒 |

### 歌词编辑器

鹿音盒内置功能完善的歌词编辑器，支持从零制作或修改现有歌词：

#### 时间轴打点
选中一行歌词后，点击「打点」按钮，将当前播放位置作为该行歌词的开始时间戳插入。

#### 批量打点
从选中的行开始，每隔 3 秒自动为后续各行依次递增打点，快速完成整首歌的时间轴制作。

#### 逐字歌词制作
- 将一行歌词拆分为单个汉字/单词
- 为每个字单独打上时间戳
- 最终生成卡拉OK式逐字高亮的 KRC 歌词

#### 歌词合并与分割
- **分割** — 将一行歌词拆分为两行
- **合并** — 将相邻的两行歌词合并为一行

#### 撤销 / 重做
支持最多 50 步操作历史记录，随时撤销误操作或重做已撤销的操作。

#### 导出
编辑完成后可将歌词导出为 `.lrc` 或 `.qrc` 格式。

#### 预览应用
在编辑过程中可以实时预览歌词效果，确认无误后再保存。

### 歌词库管理

鹿音盒内置完整的歌词库管理系统：

- **分类筛选** — 按歌手、专辑、语言等维度分类浏览
- **搜索** — 通过歌名、歌手、专辑关键词快速查找
- **自动匹配** — 播放歌曲时自动从歌词库查找匹配的歌词
- **导出** — 将库中歌词导出为 .lrc 文件
- **内嵌标记** — 将歌词信息写入音频文件的 ID3 标签
- **删除管理** — 批量删除不需要的歌词条目

## 7. 音效与音质设置

### EQ 均衡器

内置 10 段图形化均衡器，覆盖从低频（60Hz）到超高频（16kHz）的完整频段调节。提供 25 种以上预设方案，涵盖人声、低频、场景、风格等类别，也可完全自定义调节。

### 空间音效增强

提供多种空间音效处理功能，包括智能去噪、夜间防惊吓等实用音效，以及低音增强、高音增强、混响、清晰度等独立调节选项，满足不同听音偏好。

### 空间音频

> 要求：Android 13 及以上系统

利用系统 Spatializer API 进行空间化音频处理，带来环绕立体声体验。支持头部追踪功能（需硬件配合）。

### 高级音频设置

面向发烧友的专业音频选项：

- **音频源码独占输出** — 绕过系统混音，直接输出原始音频数据
- **数字滤波器** — 选择不同的数字滤波算法影响音色走向
- **Dither 噪声塑形** — 减少量化失真，提升动态表现
- **交叉馈送** — 模拟扬声器串扰效应，改善耳机听感
- **前级防爆缓冲** — 防止突发大信号造成的爆音削波

### 音量控制

独立的软件内音量调节，不影响系统媒体音量设置。

### 播放速度调节

支持 0.5x 至 2.0x 的变速播放，可用于：

- 慢速练习乐器或学唱（0.5x ~ 0.9x）
- 正常速度欣赏（1.0x）
- 快速浏览内容（1.1x ~ 2.0x）

## 8. 锁屏海报功能

鹿音盒可以在锁屏界面显示精美的歌曲海报，让您的锁屏也充满音乐氛围。

### 开关控制

在设置中可以随时开启或关闭锁屏海报功能。

### 旋转海报模式

- **形状**：圆形海报
- **动画**：播放状态下每 20 秒完成一圈平滑旋转；暂停时旋转停止

### 静态海报模式

- **形状**：方形圆角卡片
- **动画**：无旋转动画，简洁稳重

### 自动加载封面

两种模式下均会自动加载当前播放歌曲的专辑封面图作为海报内容。

## 9. 本地音乐管理

### 设备扫描

点击扫描按钮，应用将自动搜索设备存储中的所有音频文件（支持 mp3、flac、wav、ogg、m4a、aac 等常见格式），并将结果整理展示。

### 排序功能

仿照电子表格的排序体验，点击任意列标题即可按该字段排序：

| 可排序列 | 说明 |
|----------|------|
| 文件夹 | 按所在文件夹分组排列 |
| 标题 | 按歌曲名字典序排列 |
| 时长 | 按歌曲长度排列 |
| 修改时间 | 按文件最后修改时间排列 |
| 添加时间 | 按添加到应用的先后顺序排列 |
| 年份 | 按歌曲发行年份排列 |

每列均支持升序和降序切换。

### 文件大小显示

直观显示每个音频文件的占用空间，单位自动适配 MB 或 KB。

### 歌曲完整性检测

自动检测每首歌的健康状态，缺失或损坏的歌曲会被明确标记。

### 批量操作

- **全选** — 一键选中当前列表全部歌曲
- **反选** — 反转当前的选中状态
- **批量删除** — 一次性删除多首歌曲
- **批量移动排序** — 拖动多首歌曲调整在列表中的位置

### 本地歌单管理

- **创建** — 新建空白本地歌单
- **编辑** — 修改歌单名称等信息
- **删除** — 不再需要的歌单可以删除
- **添加歌曲** — 从本地音乐库向歌单添加歌曲

### 元数据编辑

对本地音频文件的标签信息进行编辑，包括歌曲名、艺术家、专辑名、封面图片、歌词标签。

### 在线匹配歌词和封面

对于缺少歌词或封面的本地歌曲，可以通过在线搜索自动匹配并补全相关信息。

## 10. 下载功能

### 下载开关

全局控制是否启用歌曲下载功能。

### 下载路径设置

自定义下载文件的保存目录，默认保存在应用专属目录中。

### 文件命名格式

提供三种命名模板供选择：

| 格式 | 示例 |
|------|------|
| 歌名-歌手 | `晴天-周杰伦.mp3` |
| 歌手-歌名 | `周杰伦-晴天.mp3` |
| 歌名 | `晴天.mp3` |

### 附加选项

下载时可勾选以下附加内容：

- **下载歌词文件** — 同步下载 .lrc 歌词文件
- **罗马音歌词** — 下载带罗马音注音的歌词
- **内嵌歌词** — 将歌词写入音频文件的标签中
- **写入标签** — 写入 ID3 等标准元数据标签
- **写入封面** — 将专辑封面嵌入音频文件
- **写入别名** — 写入歌曲别名信息

### 下载历史记录

记录所有已完成和进行中的下载任务，方便查看和管理。

### 批量下载

对整个歌单执行批量下载，或手动勾选想要下载的具体歌曲后统一下载。

## 11. 数据同步功能

### WebDAV 同步

支持通过 WebDAV 协议将应用数据同步到远程服务器，兼容各类 WebDAV 服务。

### 列表同步

- **合并模式** — 远程数据和本地数据合并，保留双方内容
- **覆盖模式** — 以一端数据为准，完全替换另一端数据

### 连接历史

记录所有历史同步连接信息，方便快速重新连接常用服务器。

## 12. 设置系统详解

### 基本设置

#### 语言设置
支持中文简体、繁体及英文等多种语言界面。

#### 字体大小
提供六档可选：较小 → 小 → 标准 → 大 → 较大 → 非常大

#### 主题颜色
共 16 种精心设计的主题配色可供选择。

#### 主题模式
- **亮模式** — 明亮的浅色界面
- **暗模式** — 护眼的深色界面
- **跟随系统** — 自动随系统深色模式开关切换

#### 动态背景与自定义背景
- **动态背景** — 使用预设的动态壁纸
- **自定义背景图片** — 从相册选择任意图片作为背景
- **透明度** — 调节背景层的不透明度
- **模糊度** — 调节背景层的模糊程度

## 13. 其他功能

### 动态岛 / 原子岛通知

适配部分品牌手机的灵动岛/原子岛特性，在岛型区域实时显示当前播放信息和控制按钮，无需下拉通知栏即可操控播放。

### 音乐桌面小组件

提供 Android 桌面小组件（Widget），直接在桌面上显示播放控制和歌曲信息，无需打开应用。

### 歌手详情页

展示歌手的详细资料，包括基本信息、热门歌曲列表、发行专辑列表、相似歌手推荐。

### 专辑详情页

展示专辑的完整信息，包括专辑名称、封面、发行信息、完整曲目列表。

### 相似歌曲推荐

基于当前播放歌曲的风格特征，智能推荐相似的其他歌曲，帮助发现更多好音乐。

### 双击退出确认

连续两次点击返回键才退出应用，防止误触导致意外退出。

## 14. 权限说明

鹿音盒在运行过程中需要申请以下权限，每项权限的具体用途如下：

| 权限名称 | 用途说明 | 必要性 |
|----------|----------|--------|
| **存储权限（读写）** | 扫描和读取设备中的本地音乐文件；保存下载的歌曲、歌词和缓存数据 | 必要 — 无此权限将无法使用本地音乐和下载功能 |
| **通知权限** | 显示常驻通知栏，提供播放控制快捷操作；接收应用通知消息 | 必要 — 无此权限将无法使用通知栏控制 |
| **悬浮窗权限（Overlay）** | 在其他应用上层显示桌面歌词悬浮窗 | 可选 — 仅使用桌面歌词时需要 |
| **电池优化白名单** | 申请加入系统电池优化白名单，防止系统在后台强制终止播放进程 | 建议 — 加入后可确保后台稳定播放不被系统杀死 |

> 所有权限均遵循最小必要原则，仅在对应功能被使用时才会实际调用。
