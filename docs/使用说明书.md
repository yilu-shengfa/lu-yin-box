## 3. 快速入门指南

### 第一步：安装应用

从可信渠道下载并安装鹿音盒 APK 安装包。首次启动时，请允许应用请求的必要权限（存储权限、通知权限等），以确保各项功能正常运行。

### 第二步：导入或搜索音乐

1. 打开应用后，您可以：
   - 点击顶部的「立即搜索」入口搜索您已拥有的合法音乐资源
   - 或通过「本地音乐」功能扫描设备中的本地音频文件

2. 选择音乐文件或搜索结果，点击即可开始播放

### 第三步：探索更多功能

通过左侧导航菜单（横屏模式）或顶部导航栏（竖屏模式），您可以访问：

- **在线歌单** — 浏览和管理您的歌单收藏
- **排行榜单** — 查看音乐排行信息
- **每日推荐** — 根据您的喜好获取个性化推荐
- **本地音乐** — 扫描并播放手机中的本地音乐文件

### 第四步：个性化设置

点击导航菜单中的「软件设置」，您可以：

- 切换主题颜色和外观模式
- 调整字体大小以适应阅读习惯
- 配置桌面歌词、音效等高级功能
- 管理下载路径和数据同步

### 第五步：享受音乐

现在您已经掌握了基本用法！在播放详情页面可以查看歌词、调整音效、切换播放模式等。尽情享受音乐吧！

> **提醒**：请确保您播放的音乐内容均已获得合法授权。

## 5. 音乐播放功能

### 音乐源说明

鹿音盒支持多种方式获取和播放音乐：

- **本地音乐播放** — 播放设备存储中的本地音频文件（推荐）
- **自定义音源** — 用户可配置合法的音乐源（详见第14章）
- **通用搜索功能** — 支持搜索功能查找音乐资源

> ⚠️ **重要提示**：
> - 使用任何在线音乐源前，请确保您有权访问和使用该音乐内容
> - 请尊重各音乐服务提供商的服务条款和版权政策
> - 建议优先使用本地音乐功能，确保内容合法性
> - 不同音源可用的音质选项可能有所不同

### 播放控制

| 功能 | 操作方式 |
|------|----------|
| **播放 / 暂停** | 点击底部播放栏中央按钮或播放页封面区域 |
| **上一曲 / 下一曲** | 点击播放控制区的上/下一曲按钮 |
| **进度条拖拽** | 左右拖拽进度条跳转到指定播放位置 |
| **定时停止** | 设置定时器，到时自动停止播放（支持多种时长选择） |

### 播放模式

支持以下六种播放模式，点击循环模式按钮依次切换：

1. **列表循环** — 播放完列表最后一首后回到第一首继续播放
2. **顺序播放** — 按列表顺序依次播放，播完即停
3. **随机播放** — 随机打乱顺序播放
4. **单曲循环** — 反复播放当前这一首歌曲
5. **禁用切换** — 锁定当前播放模式，防止误触切换
6. **心动模式** — 智能优先播放您标记为"喜欢"的歌曲

### 音质选择

鹿音盒提供多档音质供您选择：

| 音质等级 | 说明 |
|----------|------|
| **128K** | 标准 MP3 音质，节省流量和存储空间 |
| **320K** | 高品质 MP3 音质，适合大多数场景 |
| **FLAC** | 无损压缩格式，保留完整音质细节 |
| **Hires 无损 24-Bit** | 高解析度无损音质，Hi-Fi 级体验 |
| **臻品音质** | 高码率音质选项 |
| **臻品音质 2.0** | 升级版高码率音质 |
| **臻品母带** | 录音室母带级原始音质，极致还原 |

> 注意：高音质选项需要对应音乐源支持，且会消耗更多网络流量和存储空间。

### 其他播放功能

- **歌曲换源** — 当前歌曲无法播放时，一键切换到其他可用音源尝试播放
- **播放历史** — 记录最近播放过的歌曲，方便回溯收听
- **评论查看与发送** — 查看其他用户的评论，也可以发表自己的评论
- **MV 播放** — 支持观看歌曲的 MV 视频（如有）
- **分享歌曲** — 将喜欢的歌曲分享给好友
- **稍后播放** — 将歌曲加入"稍后播放"队列，当前歌曲播完后自动播放
- **不喜欢** — 标记不喜欢的歌曲，系统将减少同类推荐
- **上下滑动切歌** — 在播放详情页上下滑动屏幕即可快速切换歌曲

## 6. 歌词功能详解

鹿音盒拥有强大而精美的歌词系统，满足从日常欣赏到专业制作的各种需求。

### 桌面歌词悬浮窗

桌面歌词可在任何应用上层显示，让您在使用其他应用时也能看到当前播放歌曲的歌词：

| 设置项 | 说明 |
|--------|------|
| **字体大小** | 调整悬浮窗歌词的文字大小 |
| **透明度** | 调整悬浮窗背景的透明程度 |
| **位置** | 自由拖拽调整悬浮窗在屏幕上的位置 |
| **最大行数** | 控制同时显示的歌词行数 |
| **单行模式** | 仅显示当前正在演唱的一行歌词 |
| **锁定** | 锁定位置后防止误触移动 |
| **主题色** | 选择歌词的高亮主题颜色 |

### 播放页歌词

播放详情页内置两种歌词布局方式：

#### 水平布局

- 歌词与封面左右并列显示
- 适合横屏使用，信息一目了然

#### 垂直布局

- 歌词与封面上下排列显示
- 适合竖屏手持操作

**可调参数：**

- **字体大小** — 调整歌词文字大小
- **对齐方式** — 左对齐 / 居中对齐 / 右对齐
- **宽度调节** — 在水平布局下调整歌词区域的宽度占比

### KRC/KSC 逐字歌词

支持 KRC 和 KSC 格式的逐字时间轴歌词，呈现卡拉OK式的逐字高亮效果——每个字随着演唱进度逐个变色点亮，带来沉浸式跟唱体验。

### 双语歌词

同时显示原文歌词和翻译歌词，帮助理解外语歌曲的含义。支持自动匹配翻译歌词或手动导入。

### 罗马音歌词

为日语、韩语等非拉丁语系歌曲显示罗马拼音注音，方便学唱外语歌曲。

### 自定义歌词背景

让您的歌词显示更加个性化：

- **本地图片** — 从相册中选择本地图片作为歌词背景
- **网络图片** — 输入图片 URL 使用网络图片作为背景
- **透明度** — 调节背景图的透明程度
- **模糊度** — 调节背景图的模糊效果（毛玻璃效果）

### 歌词高亮颜色定制

自由选择当前演唱行的文字高亮颜色，支持多种预设色和自定义颜色。

### 蓝牙歌词

支持通过蓝牙将歌词推送到车载音响、智能手表等支持蓝牙歌词显示的设备上。

### 歌词元信息过滤

自动过滤歌词文件中的元数据标签（如 `[ti:]`、`[ar:]`、`[al:]` 等），只保留纯净的歌词正文内容。

### 歌词拖拽调整进度

在播放详情页长按歌词区域并左右拖拽，可以直接调整播放进度——拖到哪句歌词就从哪里开始播放。

## 7. 歌词高级功能

### 导入本地歌词

支持通过 SAF（存储访问框架）文件选择器导入以下格式的本地歌词文件：

- `.lrc` — 标准 LRC 格式歌词
- `.krc` — KRC 加密格式歌词
- `.txt` — 纯文本歌词
- `.trc` — 其他歌词格式

### 智能自动校正

导入歌词后，系统会自动检测歌词的时间标签是否存在偏移（即歌词显示快了或慢了），并给出修正建议。您可以选择接受建议或手动微调。

### 手动调整偏移

提供五档精细偏移调节：

| 按钮 | 效果 |
|------|------|
| **-1s** | 歌词整体提前 1 秒 |
| **-100ms** | 歌词整体提前 100 毫秒 |
| **重置** | 恢复默认无偏移状态 |
| **+100ms** | 歌词整体延后 100 毫秒 |
| **+1s** | 歌词整体延后 1 秒 |

### 歌词编辑器

鹿音盒内置功能完善的歌词编辑器，支持从零制作或修改现有歌词：

#### 时间轴打点

选中一行歌词后，点击「打点」按钮，将当前播放位置作为该行歌词的开始时间戳插入。

#### 批量打点

从选中的行开始，每隔 3 秒自动为后续各行依次递增打点，快速完成整首歌的时间轴制作。

#### 逐字歌词制作

- 将一行歌词拆分为单个汉字/单词
- 为每个字单独打上时间戳
- 最终生成卡拉OK式逐字高亮的 KRC 歌词

#### 歌词合并与分割

- **分割** — 将一行歌词拆分为两行
- **合并** — 将相邻的两行歌词合并为一行

#### 撤销 / 重做

支持最多 50 步操作历史记录，随时撤销误操作或重做已撤销的操作。

#### 导出

编辑完成后可将歌词导出为：

- `.lrc` — 标准 LRC 格式
- `.qrc` — 应用专用格式

#### 绑定歌曲

将编辑好的歌词与歌曲绑定，支持通过以下方式进行匹配：

- 歌曲 ID（songId）
- 文件名匹配
- 歌名匹配

#### 预览应用

在编辑过程中可以实时预览歌词效果，确认无误后再保存。

### 歌词库管理

鹿音盒内置完整的歌词库管理系统：

- **分类筛选** — 按歌手、专辑、语言等维度分类浏览
- **搜索** — 通过歌名、歌手、专辑关键词快速查找
- **自动匹配** — 播放歌曲时自动从歌词库查找匹配的歌词
- **导出** — 将库中歌词导出为 .lrc 文件
- **内嵌标记** — 将歌词信息写入音频文件的 ID3 标签
- **删除管理** — 批量删除不需要的歌词条目

## 8. 音效与音质设置

### EQ 均衡器

内置 10 段图形化均衡器，覆盖从低频（60Hz）到超高频（16kHz）的完整频段调节。

**25+ 预设方案包括但不限于：**

| 类别 | 预设示例 |
|------|----------|
| 人声类 | 人声增强、清晰对白 |
| 低频类 | 重低音、强劲低音 |
| 场景类 | 电影原声、现场演出 |
| 风格类 | 古风、国风器乐、摇滚、电子、爵士、古典 |
| 自定义 | 完全自主调节每个频段的增益 |

### SQ 超级音效

提供 17 种专业空间音效处理：

- **智能去噪** — 自动降低背景噪声
- **夜间防惊吓** — 压制突然的音量峰值
- **人声隔离** — 突出人声、弱化伴奏
- **音量突破** — 突破系统音量上限限制
- 以及更多音效方案...

### 低音增强 / 高音增强 / 混响 / 清晰度

- **低音增强** — 加强低频力度和厚度
- **高音增强** — 提升高频细节和空气感
- **混响** — 添加空间混响效果，模拟不同环境的声场
- **清晰度** — 优化声音分离度和定位感

### 空间音频

> 要求：Android 13 及以上系统

利用系统 Spatializer API 进行空间化音频处理，带来环绕立体声体验。支持头部追踪功能（需硬件配合），转动头部时声音方位随之变化，营造身临其境之感。

### 高级音频设置

面向发烧友的专业音频选项：

## 10.5 局域网音乐功能

鹿音盒支持通过局域网协议访问和播放远程设备上的音乐文件，让您轻松管理家庭 NAS、电脑共享文件夹等设备中的音乐资源。

### 支持的协议

| 协议 | 说明 | 适用场景 |
|------|------|----------|
| **SMB/CIFS** | Windows 共享文件夹协议 | 访问 Windows 电脑、NAS 的共享目录 |
| **FTP** | 文件传输协议 | 访问 FTP 服务器上的音乐文件 |
| **WebDAV** | 基于 HTTP 的文件管理协议 | 访问支持 WebDAV 的网盘/NAS |

### 功能特性

- **自动发现** — 自动扫描局域网内的 SMB 设备，一键连接
- **手动添加** — 支持手动输入地址连接 FTP/WebDAV 服务器
- **浏览播放** — 像浏览本地文件一样浏览远程目录，点击即可播放
- **下载功能** — 将远程音乐文件下载到本地设备
- **断线重连** — 网络中断后自动尝试重新连接
- **多格式支持** — 支持 MP3、FLAC、WAV、DSD（DSF/DFF）、APE 等几乎所有音频格式
- **设备智能识别** — 扫描到的设备自动识别类型（📱手机 / 💻电脑 / 📡路由器 / 🗄️NAS / 📺电视）并显示对应图标

### 使用方法

1. 在主界面导航菜单中找到「局域网音乐」入口
2. 首次使用时，应用会自动扫描局域网中的 SMB 共享设备
3. 点击发现的设备或手动添加服务器地址
4. 输入用户名密码（如需要）后建立连接
5. 浏览目录找到音乐文件，点击即可播放

> ⚠️ **提示**：局域网音乐功能仅用于访问您拥有合法使用权的音频资源。请确保您有权限访问目标设备上的文件内容。
