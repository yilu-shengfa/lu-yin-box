## 1. 软件概述

| 项目 | 信息 |
|------|------|
| **软件名称** | 鹿音盒（LuYin Music Box） |
| **当前版本** | v1.8.91 |
| **包名** | tv.ylsf.box.mobile |
| **支持平台** | Android 5.0（API 21）及以上，主要支持 arm64-v8a 架构 |

### 简介

鹿音盒是一款功能丰富的 Android 音乐播放器应用，提供高品质音乐播放、精美歌词显示、专业级音效调节、本地音乐管理、数据云端同步等全方位音乐体验。

### 核心功能亮点

- **31段专业频段调音** — 支持标准模式（10频段）/专业模式（31频段）精细调节
- **DSD 秒播** — FFmpeg 原生层直接解码 DSD 文件，无需预转换
- **全系耳机头部追踪检测** — 支持 AirPods/Beats/华强北/恒玄/高通/络达等耳机
- **KRC 逐字歌词** — 卡拉OK式逐字高亮效果，支持动效和渐变
- **桌面歌词悬浮窗** — 完整的桌面歌词功能，支持双语/罗马音显示

## 2. 快速入门指南

### 第一步：安装应用

从 GitHub Releases 下载并安装鹿音盒 APK 安装包。首次启动时，请允许应用请求的必要权限。

### 第二步：搜索并播放音乐

1. 打开应用后，点击顶部的「立即搜索」入口
2. 输入您想听的歌曲名称或歌手名
3. 选择一个搜索结果，点击即可开始播放

### 第三步：探索更多内容

通过左侧导航菜单或顶部导航栏，您可以访问：

- **在线歌单** — 浏览热门歌单和精选推荐
- **排行榜单** — 查看各大平台音乐排行榜
- **每日推荐** — 根据您的喜好获取个性化推荐
- **本地音乐** — 扫描并播放手机中的本地音乐文件

### 第四步：个性化设置

点击导航菜单中的「软件设置」，您可以：

- 切换主题颜色和外观模式
- 调整字体大小以适应阅读习惯
- 配置桌面歌词、音效等高级功能
- 管理下载路径和数据同步

## 4. 音乐播放功能

### 播放控制

| 功能 | 操作方式 |
|------|----------|
| **播放 / 暂停** | 点击底部播放栏中央按钮或播放页封面区域 |
| **上一曲 / 下一曲** | 点击播放控制区的上/下一曲按钮 |
| **进度条拖拽** | 左右拖拽进度条跳转到指定播放位置 |
| **定时停止** | 设置定时器，到时自动停止播放 |

### 播放模式

支持以下六种播放模式，点击循环模式按钮依次切换：

1. **列表循环** — 播放完列表最后一首后回到第一首继续播放
2. **顺序播放** — 按列表顺序依次播放，播完即停
3. **随机播放** — 随机打乱顺序播放
4. **单曲循环** — 反复播放当前这一首歌曲
5. **禁用切换** — 锁定当前播放模式，防止误触切换
6. **心动模式** — 智能优先播放您标记为"喜欢"的歌曲

### 音质选择

| 音质等级 | 说明 |
|----------|------|
| **128K** | 标准 MP3 音质，节省流量和存储空间 |
| **320K** | 高品质 MP3 音质，适合大多数场景 |
| **FLAC** | 无损压缩格式，保留完整音质细节 |
| **更高音质** | 视音乐源支持情况提供更高级别的音质选项 |

## 5. 歌词功能详解

### 桌面歌词悬浮窗

桌面歌词可在任何应用上层显示，让您在使用其他应用时也能看到当前播放歌曲的歌词：

| 设置项 | 说明 |
|--------|------|
| **字体大小** | 调整悬浮窗歌词的文字大小 |
| **透明度** | 调整悬浮窗背景的透明程度 |
| **位置** | 自由拖拽调整悬浮窗在屏幕上的位置 |
| **最大行数** | 控制同时显示的歌词行数 |
| **单行模式** | 仅显示当前正在演唱的一行歌词 |
| **锁定** | 锁定位置后防止误触移动 |
| **主题色** | 选择歌词的高亮主题颜色 |

### KRC/KSC 逐字歌词

支持 KRC 和 KSC 格式的逐字时间轴歌词，呈现卡拉OK式的逐字高亮效果。

### 逐字动效与渐变

- **动效** — 逐字放大效果，让歌词更生动
- **渐变** — 模拟 PeekPro 的逐字从左到右平滑渐变效果
- 动效和渐变可同时启用，打造沉浸式听歌体验

### 3D 动效歌词

全新 3D 歌词渲染引擎，为逐字歌词注入立体动效，支持三种风格：

| 动效 | 说明 |
|------|------|
| **激光淡入** | 歌词逐字以激光光束效果淡入显示 |
| **旋转飞入** | 歌词逐字以 3D 旋转方式飞入画面 |
| **霓虹重影** | 歌词呈现霓虹灯重影发光效果 |

**两种模式：**
- **自动模式** — 跟随当前音效自动切换 3D 动效类型
- **手动模式** — 固定使用指定的 3D 动效类型

### 双语歌词

同时显示原文歌词和翻译歌词，帮助理解外语歌曲的含义。

按歌手、专辑、语言等维度分类浏览和管理歌词库。

## 7. 音效与音质设置

### EQ 均衡器

鹿音盒提供专业级的均衡器调节功能：

#### 标准模式（10频段）
覆盖从低频（60Hz）到超高频（16kHz）的完整频段调节。

#### 专业模式（31频段）
遵循 ISO 266 标准，覆盖 20Hz-20kHz 全频段，采用 1/3 Octave 间隔。

**预设方案**：提供 30+ 种预设，涵盖人声增强、古风、电影原声、摇滚、爵士、重低音、震撼DJ、HIFI发烧友等多种风格。

### 空间音效增强

提供多种空间音效处理功能，包括智能去噪、夜间防惊吓、低音增强、高音增强、混响、清晰度等独立调节选项。

### 空间音频

利用系统 Spatializer API 进行空间化音频处理，带来环绕立体声体验。支持头部追踪功能（需硬件配合）。

### 高级音频设置

- **音频源码独占输出** — 绕过系统混音，直接输出原始音频数据
- **数字滤波器** — 选择不同的数字滤波算法
- **Dither 噪声塑形** — 减少量化失真，提升动态表现
- **DSD 秒播** — FFmpeg 原生层直接解码 DSD 文件，无需预转换

## 8. 锁屏海报功能

### 旋转海报模式

- **形状**：圆形海报
- **动画**：播放状态下每 20 秒完成一圈平滑旋转

### 静态海报模式

- **形状**：方形圆角卡片
- **动画**：无旋转动画，简洁稳重

## 9. 本地音乐管理

### 设备扫描

自动搜索设备存储中的所有音频文件，支持 mp3、flac、wav、ogg、m4a、aac、dsf、dff 等常见和高解析格式。

### 局域网音乐（SMB 共享）

支持浏览并播放局域网中 Windows 共享文件夹或 NAS 设备上的音频文件。

### 排序功能

点击任意列标题即可按该字段排序，支持升序和降序切换。

### 批量操作

- **全选** — 一键选中当前列表全部歌曲
- **反选** — 反转当前的选中状态
- **批量删除** — 一次性删除多首歌曲

## 10. 下载功能

### 下载路径设置

自定义下载文件的保存目录。

### 文件命名格式

提供三种命名模板供选择：歌名-歌手、歌手-歌名、歌名。

### 附加选项

下载时可勾选下载歌词文件、罗马音歌词、内嵌歌词、写入标签、写入封面、写入别名等选项。

### 批量下载

对整个歌单执行批量下载，或手动勾选想要下载的具体歌曲后统一下载。

## 12. 设置系统详解

### 基本设置

- **语言设置** — 支持中文简体、繁体及英文等多种语言
- **字体大小** — 提供六档可选
- **主题颜色** — 共 16 种精心设计的主题配色
- **主题模式** — 亮模式、暗模式、跟随系统

### 播放设置

| 设置项 | 说明 |
|--------|------|
| 默认播放音质 | 新歌首次播放时的默认音质等级 |
| 记住播放进度 | 关闭重开后从上次位置继续播放 |
| 显示歌词翻译 | 播放页是否同时显示翻译歌词 |
| 显示逐字歌词 | 是否启用卡拉OK逐字效果 |
| 蓝牙歌词 | 是否开启蓝牙歌词推送 |
| 通知栏显示封面 | 通知栏是否显示专辑封面图 |

### 搜索设置

| 设置项 | 说明 |
|--------|------|
| 显示热门搜索 | 搜索框下方是否展示当前热搜关键词 |
| 显示历史搜索记录 | 是否保留和显示过往搜索记录，最多保留 15 条 |
| 自动换源搜索 | 当前音源搜不到结果时自动切换到其他音源继续搜索 |
| 默认搜索来源 | 设置默认使用的搜索音源 |
| 搜索结果显示来源标签 | 在搜索结果中显示每首歌曲所属的音源标签 |
| 搜索结果全部播放 | 在搜索结果页面显示「全部播放」按钮，一键播放当前搜索结果中的全部歌曲 |

### 歌词设置

#### 桌面歌词
- 开关总控制
- 锁定位置
- 样式配置（字体、颜色、透明度、位置等）
- 逐字歌词、动效、渐变设置
- 显示歌词翻译、显示歌词罗马音

#### 播放页歌词
- 字体大小
- 对齐方式（左/中/右）
- 宽度比例
- 背景样式

### 权限管理

| 权限 | 用途 |
|------|------|
| 存储权限 | 读写本地音乐文件和下载目录 |
| 通知权限 | 显示播放通知和控制通知栏 |

## 13. 其他功能

### 动态岛 / 原子岛通知

适配部分品牌手机的灵动岛/原子岛特性，在岛型区域实时显示当前播放信息和控制按钮。

### 音乐桌面小组件

提供 Android 桌面小组件，直接在桌面上显示播放控制和歌曲信息。

### 歌手详情页

展示歌手的详细资料，包括基本信息、热门歌曲列表、发行专辑列表、相似歌手推荐。

### 专辑详情页

展示专辑的完整信息，包括专辑名称、封面、发行信息、完整曲目列表。

## 14. 权限说明

| 权限名称 | 用途说明 | 必要性 |
|----------|----------|--------|
| **存储权限（读写）** | 扫描和读取设备中的本地音乐文件；保存下载的歌曲、歌词和缓存数据 | 必要 |
| **通知权限** | 显示常驻通知栏，提供播放控制快捷操作；接收应用通知消息 | 必要 |
| **悬浮窗权限（Overlay）** | 在其他应用上层显示桌面歌词悬浮窗 | 可选 |
| **电池优化白名单** | 申请加入系统电池优化白名单，防止系统在后台强制终止播放进程 | 建议 |
