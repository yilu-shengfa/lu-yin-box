# 快速入门指南

> 从安装到畅听，5 分钟上手鹿音盒

## 1. 下载与安装

1. 前往 [Releases 页面](https://github.com/yilu-shengfa/lu-yin-box/releases)
2. 下载对应您设备架构的 APK 文件（推荐 `arm64-v8a`）
3. 在设备上安装 APK（可能需要开启"允许安装未知来源应用"）
4. 首次启动，授予必要的存储和通知权限

## 2. 第一次使用

启动后，您会看到主界面。顶部是搜索栏，左侧（横屏）或顶部（竖屏）是导航菜单。

### 搜索音乐

点击「立即搜索」，输入歌名或歌手名，即可看到各平台搜索结果。

### 播放歌曲

点击任意搜索结果即可播放。底部会出现播放栏，显示当前歌曲信息和控制按钮。

## 3. 探索功能

### 查看歌词

点击底部播放栏进入播放详情页，上下滑动可在"海报页"和"歌词页"间切换。

### 调整音效

在播放详情页或设置中进入音效调节，使用 EQ 均衡器、空间音效等功能。

### 收藏歌单

点击歌曲右侧的菜单按钮，可以收藏到自己的歌单，或标记喜欢的歌曲。

## 4. 个性化设置

进入「软件设置」→「基本设置」，您可以：

- 切换**主题颜色**（16 种可选）
- 调整**字体大小**
- 设置**桌面歌词**样式
- 配置**动态背景**

## 5. 小贴士

- 💡 **上下滑动切歌** — 在播放页面上下滑动即可切换歌曲
- 💡 **长按歌词调进度** — 在歌词页长按并左右拖拽可调整播放位置
- 💡 **桌面歌词** — 开启后可在其他应用上层显示歌词
- 💡 **定时停止** — 设置定时器，到时间自动停止播放
