## 安装应用

### 步骤 1：下载 APK

从 GitHub Releases 页面下载最新版本的 APK 文件。根据你的设备选择合适的架构：

| 架构 | 适用设备 |
|------|----------|
| `arm64-v8a` | 主流 Android 手机/平板（推荐） |
| `armeabi-v7a` | 旧款 32 位 Android 设备 |
| `x86_64` | Android 模拟器（64位） |
| `x86` | Android 模拟器（32位） |

### 步骤 2：安装应用

1. 在手机上找到下载的 APK 文件
2. 点击文件开始安装
3. 如果系统提示「未知来源」，请在设置中允许安装

## 首次启动

### 权限请求

首次启动时，应用会请求以下权限：

- **存储权限** — 用于扫描本地音乐文件和保存下载内容
- **通知权限** — 用于显示播放通知和控制通知栏
- **悬浮窗权限** — 用于显示桌面歌词（可选）

### 欢迎界面

首次启动会显示欢迎界面，引导你了解应用的基本功能。

## 搜索音乐

### 基本搜索

1. 点击顶部搜索栏
2. 输入歌曲名称或歌手名
3. 点击搜索按钮或按回车键

### 历史搜索记录

开启后，搜索框下方会显示最近搜索过的关键词（最多 15 条），点击即可快速再次搜索。

### 全部播放

开启「搜索结果全部播放」后，搜索结果页面顶部会出现「全部播放」按钮，一键播放全部搜索结果。

### 自动换源搜索

开启后，当前音源搜不到结果时会自动切换到其他音源继续搜索，提升搜索命中率。

### 搜索结果

搜索结果会显示：
- 歌曲列表
- 歌手列表
- 专辑列表

### 选择音源

不同音源可用的音质选项可能有所不同，点击歌曲右侧菜单可以切换音源。

### 搜索设置

在「软件设置 → 搜索设置」中可配置：
- 显示热门搜索
- 显示历史搜索记录
- 自动换源搜索
- 默认搜索来源
- 搜索结果显示来源标签
- 搜索结果全部播放

## 播放控制

### 播放/暂停

- 点击底部播放栏中央按钮
- 点击播放页封面区域

### 切歌

- 点击底部播放栏的上/下一曲按钮
- 在播放详情页上下滑动屏幕

### 进度控制

- 左右拖拽进度条跳转到指定位置

### 播放模式

点击底部播放栏的循环模式按钮，依次切换：
- 列表循环
- 顺序播放
- 随机播放
- 单曲循环
- 禁用切换
- 心动模式

## 歌词设置

### 开启逐字歌词

1. 进入播放详情页
2. 点击右上角设置按钮
3. 开启「显示逐字歌词」选项

### 动效与渐变

- **动效** — 逐字放大效果
- **渐变** — 逐字从左到右平滑渐变效果
- 两者可同时启用

### 3D 动效歌词

开启「3D 动效」后，歌词将以立体动效呈现，支持激光淡入、旋转飞入、霓虹重影三种效果。可选择自动模式（跟随音效自动切换）或手动模式（固定指定动效）。

### 双语歌词

开启「显示歌词翻译」选项，同时显示原文和翻译。

## 音效调节

### EQ 均衡器

1. 进入播放详情页
2. 点击音效按钮
3. 选择标准模式（10频段）或专业模式（31频段）
4. 选择预设方案或自定义调节

### 预设方案

提供 30+ 种预设：
- 人声增强、古风、电影原声
- 摇滚、爵士、重低音
- 震撼DJ、HIFI发烧友等

## 本地音乐

### 扫描本地音乐

1. 进入「本地音乐」页面
2. 点击扫描按钮
3. 等待扫描完成

### 支持格式

支持 mp3、flac、wav、ogg、m4a、aac、dsf、dff、ape 等 70+ 种音频格式。

### 批量操作

- 全选：一键选中所有歌曲
- 反选：反转选中状态
- 批量删除：删除选中的歌曲

## 视频播放

### 本地视频

1. 进入「本地视频」页面
2. 点击「扫描」按钮搜索设备中的视频文件
3. 点击视频即可开始播放

### 在线MV

1. 进入「在线MV」页面
2. 在搜索栏输入关键词（歌曲名/歌手名/专辑名）
3. 点击搜索结果播放

### 视频播放控制

- **点击屏幕中间** — 显示/隐藏控制栏
- **左侧上下滑动** — 调节屏幕亮度
- **右侧上下滑动** — 调节系统音量
- **画面比例** — 支持原始比例、适应屏幕、裁剪填充、拉伸填充
- **画中画** — 点击画中画按钮，视频悬浮播放
- **播放列表** — 点击播放列表按钮，右侧滑出队列

### 视频管理

- **排序** — 按名称、时间、大小排序
- **筛选** — 关键词筛选视频
- **批量操作** — 全选/反选/添加/删除
- **视频表单** — 创建歌单管理视频

## 有声书

### 浏览有声书

1. 进入「本地音乐」页面
2. 切换到「有声书」标签页
3. 应用会自动检测本地音乐文件夹中的有声书文件

### 有声书管理

- **切换显示模式** — 点击右上角图标切换封面/数字列表模式
- **排序** — 点击「排序」按钮，可按书名、时长、大小排序
- **筛选** — 在关键词输入框中输入文字筛选
- **添加** — 点击「添加」按钮手动标记歌曲为有声书

## A-B 复读

### 使用方法

1. 在播放页面点击「A-B」按钮
2. 在需要循环的起点点击，设定 A 点
3. 在需要循环的终点再次点击，设定 B 点
4. 音频会在 A、B 点之间自动循环播放
5. 再次点击「A-B」按钮可取消复读
