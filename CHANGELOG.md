# 更新日志

---

### v1.8.95（2026-07-23）

#### 新增功能
- **回放增益（ReplayGain）** — 基于 EBU R128 标准自动统一不同歌曲的感知响度，首次播放时自动分析响度并缓存结果，消除手动调音量的需要
- **有声书识别** — 自动检测本地音乐中的有声书文件，支持封面/数字列表切换、排序筛选（书名/时长/大小）、关键词筛选、手动标记添加
- **A-B 复读** — 支持设置 A/B 点标记实现区间循环播放，适用于语言学习和歌词练习场景

#### 功能改进
- **歌单歌曲工具栏** — 修复歌单歌曲 Tab 工具栏行不显示的问题，临时列表也支持排序、关键词筛选、列表模式切换
- **本地音乐 UI 优化** — 全部歌曲数量移至「检测」按钮右侧显示；有声书删除重复的数量行，改用工具栏内显示
- **有声书添加图标** — 修复有声书添加按钮图标显示为问号的问题

#### 问题修复
- **抽屉弹出修复** — 修复 TagList 组件条件渲染导致 DrawerLayoutAndroid 抽屉无法弹出的问题
- **导航菜单修复** — 修复搜索→歌手→播放→返回后，导航菜单抽屉因布局循环无法弹出的问题
- **音效杂音修复** — 修复拖动进度条和切换音效时 FFmpeg filter graph 状态清零导致"呲"声杂音的问题
- **进度条闪退修复** — 修复 JS 线程和音频线程同时操作 FFmpeg filter context 导致 SIGSEGV 崩溃的问题
- **EACCES 权限修复** — 修复 Android 36 设备上运行时权限未授予导致无法播放本地音乐的问题
- **有声书识别优化** — 修复非有声书内容被误识别为有声书的问题
- **A-B 文字颜色** — 修复 A-B 复读激活后文字颜色与背景颜色重合导致不可见的问题

### v1.8.93（2026-07-23）

#### 新增功能
- **自定义字体上限提升** — 自定义字体导入上限从 6 个提升至 100 个，支持 MD5 去重检测
- **本地音乐 Tab 改名** — 「正在播放」Tab 更名为「歌单歌曲」，更符合实际功能

#### 功能改进
- **临时列表统一播放管理** — 重构临时列表与本地音乐、在线音乐、排行榜单的关系，临时列表作为统一播放队列，不再持久化存储，减少存储压力
- **歌词缓存优化** — 歌词缓存从 AsyncStorage 迁移至文件存储，支持 LRU 淘汰策略（默认 5000 首），彻底解决 AsyncStorage SQLITE_FULL 错误
- **歌词缓存文件名修复** — 使用 MD5 哈希作为缓存文件名，解决 ENAMETOOLONG（文件名过长）错误
- **本地音乐排序持久化** — 正在播放列表排序后自动保存，重启软件后保持排序状态
- **数据持久化增强** — 修复软件彻底退出或重启后，下载记录、自定义源设置及菜单显示设置丢失的问题
- **本地音乐跨 Tab 同步** — 临时列表播放本地歌曲时，歌单歌曲和全部歌曲界面同步高亮当前播放位置

#### 问题修复
- **修复歌单歌曲移动崩溃** — 修复歌单内移动歌曲顺序后 setForceUpdate 未定义导致的运行时错误
- **修复歌曲列表滚动位置** — 修复点击歌曲后列表意外返回顶部的问题
- **错误日志优化** — 减少不必要的错误日志向用户展示，仅保留用户可理解的错误信息

### v1.8.92（2026-07-14）

#### 新增功能
- **视频播放功能** — 全面支持本地视频和在线MV播放，内置播放控制栏
- **视频画面比例设置** — 支持原始比例、适应屏幕、裁剪填充、拉伸填充四种模式
- **视频手势控制** — 左侧滑动调节亮度，右侧滑动调节音量
- **画中画（PIP）** — 支持悬浮小窗播放
- **视频全屏播放** — 横屏全屏播放，覆盖摄像头挖孔区域
- **本地视频管理** — 扫描、排序、筛选、批量操作
- **视频表单管理** — 创建、编辑、管理视频歌单
- **在线MV搜索** — 支持按歌曲名、歌手名、专辑名搜索

#### 功能改进
- **音频爆音修复** — 修复播放开始时突然爆音问题
- **性能大幅优化** — 优化列表渲染性能，动画时长缩短至 200-300ms
- **华为系统兼容性** — 优化鸿蒙兼容层渲染性能
