# 更新日志

---

### v1.8.93（2026-07-23）

#### 新增功能
- **自定义字体上限提升** — 自定义字体导入上限从 6 个提升至 100 个，支持 MD5 去重检测
- **本地音乐 Tab 改名** — 「正在播放」Tab 更名为「歌单歌曲」，更符合实际功能

#### 功能改进
- **临时列表统一播放管理** — 重构临时列表与本地音乐、在线音乐、排行榜单的关系，临时列表作为统一播放队列，不再持久化存储，减少存储压力
- **歌词缓存优化** — 歌词缓存从 AsyncStorage 迁移至文件存储，支持 LRU 淘汰策略（默认 5000 首），彻底解决 AsyncStorage SQLITE_FULL 错误
- **歌词缓存文件名修复** — 使用 MD5 哈希作为缓存文件名，解决 ENAMETOOLONG（文件名过长）错误
- **本地音乐排序持久化** — 正在播放列表排序后自动保存，重启软件后保持排序状态
- **数据持久化增强** — 修复软件彻底退出或重启后，下载记录、自定义源设置及菜单显示设置丢失的问题
- **本地音乐跨 Tab 同步** — 临时列表播放本地歌曲时，歌单歌曲和全部歌曲界面同步高亮当前播放位置

#### 问题修复
- **修复歌单歌曲移动崩溃** — 修复歌单内移动歌曲顺序后 setForceUpdate 未定义导致的运行时错误
- **修复歌曲列表滚动位置** — 修复点击歌曲后列表意外返回顶部的问题
- **错误日志优化** — 减少不必要的错误日志向用户展示，仅保留用户可理解的错误信息

### v1.8.92（2026-07-14）

#### 新增功能
- **视频播放功能** — 全面支持本地视频和在线MV播放，内置播放控制栏
- **视频画面比例设置** — 支持原始比例、适应屏幕、裁剪填充、拉伸填充四种模式
- **视频手势控制** — 左侧滑动调节亮度，右侧滑动调节音量
- **画中画（PIP）** — 支持悬浮小窗播放
- **视频全屏播放** — 横屏全屏播放，覆盖摄像头挖孔区域
- **本地视频管理** — 扫描、排序、筛选、批量操作
- **视频表单管理** — 创建、编辑、管理视频歌单
- **在线MV搜索** — 支持按歌曲名、歌手名、专辑名搜索

#### 功能改进
- **音频爆音修复** — 修复播放开始时突然爆音问题
- **性能大幅优化** — 优化列表渲染性能，动画时长缩短至 200-300ms
- **华为系统兼容性** — 优化鸿蒙兼容层渲染性能

### v1.8.89（2026-07-07）

#### 新增功能
- **Media3 FFmpeg 原生解码链** — 全面支持 APE/DSD/WavPack/TTA/TAK 等格式
- **品牌名称统一** — "洛雪音乐助手"→"鹿音盒"
- **Hi-Res 独占输出** — 支持 192kHz/24bit 直通独立 DAC
- **64-bit 浮点全链路 DSP** — 自定义 C++ IIR 双二阶滤波器
- **Gapless 无缝播放** — Live 专辑曲目间无静默间隙
- **Crossfade 交叉淡入淡出** — 0~15秒可调节
- **实时频谱可视化** — Android Visualizer API + C++ FFT 双模式
- **音量突破系统限制** — 播放器设置音量最多到 300
