## 简介

鹿音盒是一款功能丰富的 Android 音乐播放器应用，提供高品质音乐播放、精美歌词显示、专业级音效调节、本地音乐管理、数据云端同步等全方位音乐体验。

本仓库**仅包含说明文档和发布包（APK）**，**不含任何源代码或品牌资产**。

## 深度开发说明

鹿音盒并非简单的 clone 或换皮，而是在原开源框架基础上进行了**大量深度开发与增强**，主要涉及以下方面：

### 歌词系统全面重构
- 全新设计的歌词引擎，支持 KRC/KSC 逐字歌词（卡拉OK效果）
- 逐字渐变效果：模拟 PeekPro 的逐字从左到右平滑渐变，独立控制动效与渐变开关
- 双语歌词（原文 + 翻译）与罗马音同步显示，支持独立开关
- 内置完整歌词编辑器：时间轴打点、批量打点、逐字制作、分割合并、撤销重做
- 歌词库管理系统：按歌手/专辑/语言分类筛选
- 桌面歌词悬浮窗：独立样式控制、拖拽进度、透明度调节、锁定（长按1.5秒解锁）
- 桌面歌词设置：显示歌词、显示播放器、锁定歌词、一键打开、显示歌词切换动画、逐字歌词、动效、渐变、显示歌词翻译、显示歌词罗马音——5行双排纵向对齐布局
- 蓝牙歌词推送功能
- 播放页歌词与桌面歌词样式完全同步
- 歌词左对齐/居中/右对齐功能

### 音效系统深度开发
- **31 段专业图形化均衡器**（ISO 266 标准，20Hz-20kHz，1/3 Octave），支持 10 段标准模式 / 31 段专业模式切换
- **30+ 种 EQ 预设方案**：人声增强、古风、电影原声、摇滚、爵士、重低音、震撼DJ、HIFI发烧友等
- **17 种 SQ 空间音效**：智能去噪、夜间防惊吓、人声隔离、环境增强、低音增强等
- 震撼DJ预设：深V型31段EQ曲线，80Hz +9dB、315Hz -3.5dB、10kHz +4.5dB，自动降低前级增益防爆音
- 空间音频（Android 13+ Spatializer API）
- 高级音频选项：源码输出、数字滤波器、Dither、Crossfeed
- FFmpeg anequalizer 作为默认音效引擎，备用 Android AudioFX
- 音量突破系统限制最多到 300
- 变速播放（0.5x ~ 2.0x）

### 交互与视觉增强
- 旋转海报 + 锁屏海报双模式
- 16 种主题配色，亮/暗/跟随系统三种模式
- 动态背景与自定义背景（支持透明度、模糊度调节）
- 可调节海报垂直位置（15%~70% 屏幕区域）
- 歌词对齐方式/宽度/字体/字色全自定义
- 自定义字体导入（最多 100 个）
- 横屏/竖屏自适应布局
- 通知中心下一曲按钮后增加桌面播放器开关图标

### 本地与管理功能增强
- 智能排序系统（Excel 表头式内联排序栏）
- 批量操作（全选/反选/上移下移/删除）
- 文件完整性检测（存在性检查 + 元数据损坏检测）
- 文件信息展示（大小/格式标签/路径）
- WebDAV 云端同步
- 全套备份恢复系统

### 专业级音频功能
- **Media3 FFmpeg 原生解码链** — 全面支持 APE/DSD(DSF/DFF)/WavPack/TTA/TAK 等极端无损格式
- **Hi-Res 独占输出** — AAudio/AudioTrack Direct Output，支持 192kHz/24bit 甚至 DSD DoP 直通独立 DAC 芯片
- **64-bit 浮点全链路 DSP** — 自定义 C++ IIR 双二阶滤波器，支持 TPDF/E-Weighted 噪声整形 Dither 抖动算法
- **Gapless 无缝播放** — Live 专辑曲目间无静默间隙，剩余5秒预加载下一首
- **Crossfade 交叉淡入淡出** — 0~15秒可调节
- **DSD → PCM 软解转换** — 将 DSD 实时解码为 24bit/176.4kHz~352.8kHz PCM 信号输出
- **DTS 数字影院音频格式支持**
- **实时频谱可视化** — Android Visualizer API + C++ FFT 双模式
- **全量音频格式支持**（70+ 种扩展名）

### 更多特色功能
- **局域网音乐** — 支持 SMB/FTP/WebDAV 协议访问 NAS、电脑共享等设备上的音乐资源
- **视频播放** — 支持本地视频和在线MV播放，画面比例切换、手势亮度/音量调节、画中画悬浮窗、播放列表管理
- 音乐桌面小组件
- 动态岛/原子岛通知适配
- 手机通知栏播放控制
- 定时停止播放
- 下载管理（不分页）
- 权限管理中心：独立的权限管理页面
- 未联网或未搜索到海报歌词标签时，使用默认占位海报和"暂未获取到歌词......"提示

- **局域网音乐列表项样式优化**
- **"添加到临时列表"功能修复**

#### 技术修复
- **DynamicsProcessing API 兼容性修复** — 适配 Android API 34+ (JDK 21)
