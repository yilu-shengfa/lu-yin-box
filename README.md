## 简介

鹿音盒是一款功能丰富的 Android 音乐播放器应用，提供高品质音乐播放、精美歌词显示、专业级音效调节、本地音乐管理、数据云端同步等全方位音乐体验。

## 功能特色

### 🎵 音乐播放
- 多音源搜索与播放
- 多种播放模式（列表循环、顺序、随机、单曲循环、心动模式）
- 多档音质选择
- 歌曲换源、MV 播放、播放历史

### 📝 歌词系统
- 桌面歌词悬浮窗（支持拖拽、锁定、透明度调节）
- KRC/KSC 逐字歌词卡拉OK效果
- 双语歌词 + 罗马音显示
- 内置歌词编辑器（时间轴打点、批量打点、逐字制作）
- 歌词智能校正和手动偏移微调
- 蓝牙歌词推送
- 自定义歌词背景

### 🔊 音效系统
- 10 段图形化均衡器 + 25+ 预设方案
- 空间音效增强
- 低音/高音增强、混响、清晰度
- 空间音频（Android 13+）
- 变速播放（0.5x ~ 2.0x）

### 🖼️ 界面与体验
- 16 种主题配色
- 亮/暗/跟随系统三种模式
- 动态背景与自定义背景
- 锁屏海报（旋转/静态双模式）
- 横屏/竖屏自适应
- 桌面小组件

### 💾 本地与管理
- 本地音乐扫描与管理
- 歌曲完整性检测
- 元数据编辑
- WebDAV 云端同步
- 全套备份恢复系统

### 🔧 其他
- 音乐桌面小组件
- 动态岛/原子岛通知适配
- 定时停止播放
- 不分页下载管理
