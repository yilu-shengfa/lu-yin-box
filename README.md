## 简介

鹿音盒是一款功能丰富的 Android 音乐播放器应用，提供高品质音乐播放、精美歌词显示、专业级音效调节、本地音乐管理、数据云端同步等全方位音乐体验。

本仓库仅包含**说明文档和发布包（APK）**，**不含任何源代码**。

## 深度开发说明

鹿音盒并非简单的 clone 或换皮，而是在原开源框架基础上进行了**大量深度开发与增强**，主要涉及以下方面：

### 歌词系统全面重构
- 全新设计的歌词引擎，支持 KRC/KSC 逐字歌词（卡拉OK效果）
- 双语歌词（原文 + 翻译）与罗马音同步显示
- 内置完整歌词编辑器：时间轴打点、批量打点、逐字制作、分割合并、撤销重做
- 歌词库管理系统：按歌手/专辑/语言分类筛选
- 桌面歌词悬浮窗：独立样式控制、拖拽进度、透明度调节、锁定
- 蓝牙歌词推送功能
- 播放页歌词与桌面歌词样式完全同步

### 音效系统深度开发
- **10 段图形化均衡器**：25+ 种预设方案（人声增强、古风、电影原声、摇滚、爵士等）
- **17 种 SQ 空间音效**：智能去噪、夜间防惊吓、人声隔离、环境增强、低音增强等
- 空间音频（Android 13+ Spatializer API）
- 高级音频选项：源码输出、数字滤波器、Dither、Crossfeed
- 变速播放（0.5x ~ 2.0x）

### 交互与视觉增强
- 旋转海报 + 锁屏海报双模式
- 16 种主题配色，亮/暗/跟随系统三种模式
- 动态背景与自定义背景
- 可调节海报垂直位置（15%~70% 屏幕区域）
- 歌词对齐方式/宽度/字体/字色全自定义
- 自定义字体导入（最多 6 个）
- 横屏/竖屏自适应布局

### 本地与管理功能增强
- 智能排序系统（Excel 表头式内联排序栏）
- 批量操作（全选/反选/上移下移/删除）
- 文件完整性检测（存在性检查 + 元数据损坏检测）
- 文件信息展示（大小/格式标签/路径）
- WebDAV 云端同步
- 全套备份恢复系统

### 更多特色功能
- **局域网音乐** — 支持 SMB/FTP/WebDAV 协议访问 NAS、电脑共享等设备上的音乐资源
- 音乐桌面小组件
- 动态岛/原子岛通知适配
- 手机通知栏播放控制
- 定时停止播放
- 下载管理（不分页）
